# Real-Time Chat Application

A modern, responsive chat application built with HTML, CSS, and JavaScript. Features real-time messaging, local storage persistence, and an interactive chatbot.

## Features
- Real-time messaging with smooth animations
- Messages persist using localStorage API
- Interactive chatbot with intelligent responses
- Responsive design for all devices
- Clean, modern UI with dark theme
- Sent/received message differentiation
- Chat history management

##  Tech Stack
- **HTML5** - Structure and semantics
- **CSS3** - Styling and responsive design
- **JavaScript** - Application logic and interactivity
- **localStorage API** - Client-side data persistence

## Tech Learning Takeaways

- Deepened understanding of **JavaScript event handling** and DOM manipulation
- Practiced **responsive web design** and CSS animations
- Learned to manage **client-side data persistence** using `localStorage`
